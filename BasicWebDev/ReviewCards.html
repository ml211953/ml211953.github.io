Web Design Flashcards

HTML
HyperText Markup Language, the standard markup language for creating web pages.
CSS
Cascading Style Sheets, used to describe the presentation of HTML elements.
JavaScript
A programming language that enables interactive web pages and dynamic content.
Selector
A pattern used in CSS to target the HTML elements you want to style.
Flexbox
A CSS layout module for distributing space along a single axis in a container.
Grid
A two-dimensional CSS layout system for rows and columns.
Pseudo-class
Keywords added to selectors that target elements in a specific state (e.g. :hover).
Media Query
A CSS technique to apply styles base on device characteristics like width or resolution.
